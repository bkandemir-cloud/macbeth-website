# Macbeth — Extended English framework site

Skeleton/framework site for Macbeth (macbeth.extendedenglish.com), part of
Beth's Extended English family (see the Klara, Kindertransport and Poetry
Anthology sites for the full pattern). Content to be added as source
materials become available.

## Conventions (family-wide, non-negotiable)
- Astro 5 static, GitHub → Cloudflare Pages, push to deploy.
- British English; no em dashes; never "genuine"/"genuinely".
- Footer copyright Extended English; SITE_PASSWORD middleware included (dormant).
- Copyright: short quotations only; no exam-board material reproduced.
- Device-spotting is not analysis: every technique pushed to effect.
- There is deliberately NO AI feedback tool on this site (coursework integrity rules); do not add one.
